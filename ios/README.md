# iOS MVP

This folder contains the first SwiftUI iOS and iPadOS MVP.

Goal:
- Match the current Android MVP flow.
- Compare iPad microphone recognition with the Android prototype.
- Keep the UI simple.

Included:
- Live screen
- Demo transcript playback
- Recent one-minute summary
- Current index screen with sample data
- Catchup alerts
- History
- Broadcast detail
- Microphone quality guidance
- Basic Apple Speech recognition service

How to test:
1. Open `ios/StockBroadcastCatchup.xcodeproj` in Xcode.
2. Select an iPhone or iPad device.
3. Set signing team if Xcode asks.
4. Run the app.
